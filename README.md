# Projeto onboarding



> ### Objetivo:
> Usar as funcões httpTrigger da azure para criar uma api rest simples com cobertura de testes 

### Requisitos

- Azure Functions Core Tools v3
- Nodejs 14.18 lts

### Config:

1. Criar um _.env_
2. Colocar as configurações do CosmosDB
3.
```BASH
END_POINT_URL="https://*****.documents.azure.com:****/"
DB_KEY="***********SUA KEY***********"
```

### Como usar:
```bash
$ npm i
```
```bash
$ npm start
```
